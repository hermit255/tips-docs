# 贋作

## summary
贋作（フェイク）
ランクC-40
「贋作」をNo.001から099のいずれか1枚のカードに変化させる。変身したカードは指定ポケットに入れることは出来るが、コンプリートカードの対象にはならないし、アイテム化も出来ない。カード化限界枚数にもカウントされない。
## description
贋作（フェイク）
ランクC-40
「贋作」をNo.001から099のいずれか1枚のカードに変化させる。変身したカードは指定ポケットに入れることは出来るが、コンプリートカードの対象にはならないし、アイテム化も出来ない。カード化限界枚数にもカウントされない。
